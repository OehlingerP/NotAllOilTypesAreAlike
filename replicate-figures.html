Not all oil types are alike - Replication of figures

library(NotAllOilTypesAreAlike)

This vignette is a step-by-step guide to replicate each figure of the paper “Not all oil types are alike”.

Figure 2

To prepare the data for each panel, the function merge_quality_quantity() is employed. This custom function combines trade flow data with crude oil quality information. This function includes a parameter that allows for a variation of the Russian Urals import share. As highlighted in the paper the default is 70% of all Russian imports are of type Urals. However, 70% is probably on the conservative side, as official sources say that Russian Urals is the most important type of crude oil imported from Russia, accounting for well over 80%.

df_raw <- merge_quality_quantity() %>%
  mutate(oil_type = paste(Weight, Sulfur)) %>%
  filter(!is.na(`Typical API gravity`),
         `Reporting Country` %in% get_ea_countries()) %>%
  na.omit()

Figure 3

In this figure we show the distribution of Euro Area imports for the two most common crude oil types. We do this for the whole sample period to avoid year specific variation. Dottedbars refer to medium-sour and striped bars refer to light-sweet crude oil imports. Russian imports of the two types are highlighted in red and blue, respectively.

Data preparation is straightforward and involves similar steps than in figure 2.

df_ea <- merge_quality_quantity() %>%
  filter(`Reporting Country` %in% get_ea_countries())

df_ea <- df_ea %>%
  mutate(`Reporting Country` = "Euro Area",
         date = as.numeric(substr(date, 1, 4))) %>%
  group_by(`Country of Origin`, `Type of crude oil`, `Reporting Country`,
           `date`, `Weight`, `Sulfur`) %>%
  summarize(`Volume (1000 bbl)` = sum(`Volume (1000 bbl)`, na.rm = T),
            `Total Value ($ 1000)` = sum(`Total Value ($ 1000)`, na.rm = T)) %>%
  ungroup() %>%
  mutate(`CIF price (2) ($/bbl)` = `Total Value ($ 1000)` / `Volume (1000 bbl)`,
          date = as.Date(paste0(date, "-12-31"))) %>%
  filter(`Total Value ($ 1000)` > 0, 
          `CIF price (2) ($/bbl)` > 0, 
          `Volume (1000 bbl)` > 0)
#> `summarise()` has grouped output by 'Country of Origin', 'Type of crude oil',
#> 'Reporting Country', 'date', 'Weight'. You can override using the `.groups`
#> argument.

df <- df_ea %>%
  mutate(oil_type = paste(Weight, Sulfur)) %>%
  rename("country" = `Country of Origin`) %>%
  group_by(country, oil_type) %>%
  summarize(volume = sum(`Volume (1000 bbl)`)) %>%
  filter(volume > 5000) %>%
  filter(oil_type %in% c("light sweet", "medium sour")) %>%
  pivot_wider(names_from = oil_type, values_from = volume) %>%
  pivot_longer(-country) %>%
  mutate(value = ifelse(is.na(value), 0, value)) %>%
  mutate(value = value / 1000,
         country = ifelse(country == "Russian Federation", "Russia",
                          ifelse(country == "United Kingdom", "UK",
                                 ifelse(country == "United States", "USA", 
                                        country))))
#> `summarise()` has grouped output by 'country'. You can override using the
#> `.groups` argument.

In this figure we show European crude oil imports from the United States and Russia before and after the start of the EU crude oil embargo. Panel a) Time series of imported volumes from Russia and the U.S. into the EU. Panel b) Chemical composition of crude oils imported from Russia [blue squares] in the year before the sanctions and chemical composition of U.S. imports [red bubbles] in the first year of the oil embargo. The size of the symbols refers to the volume of crude oil imports.

For this analysis we need a different dataset. We use data provide by Eurostat. At the time of writing this data contained crude oil imports into the European Union from January 2020 to October 2023 on a monthly basis. As this dataset is quite large and thus not suitable for keeping it in the package, I converted the .csv-file to .RDS which is much more memory efficient. It is still the raw data but now can be easily accessed via the package.

Panel b

In this panel we compare the quality of crude oil imports from Russia and the United States. We focus on the year before and after the embargo comes into forces. Here we should not that the crude oil embargo was implemented on December 5th, 2022. In the quality comparison we use November 2022 as the start of the sanctions which does not influence the results. It is just that at the time of writing, the last observation available was October 2023. When we use November, we can use 12 months for both periods (Russian imports before the sanctions and U.S. imports after the sanctions).

The only thing we need to take care of is the aggregation of oil quality. Here we apply our custom function mean_api() as we can not simply take the average of API gravity. We first need to convert it to specific gravity before we can take means and convert back.

df_oil_quality_ru <- df %>%
  filter(repCountryIso == "EU", sourceCountryIso %in% c("RU"),
         month <= as.Date("2022-10-31"),
         month > as.Date("2021-10-31")) %>%
  mutate(volShare = volume/sum(volume),
         meanApi = mean_api(api, volShare),
         meanSul = sum(sul*volShare)) %>%
  select(meanApi, meanSul) %>%
  unique()

df_oil_quality_us <- df %>%
  filter(repCountryIso == "EU", sourceCountryIso %in% c("US"),
         month > as.Date("2022-10-31")) %>%
  mutate(volShare = volume/sum(volume),
         meanApi = mean_api(api, volShare),
         meanSul = sum(sul*volShare)) %>%
  select(meanApi, meanSul) %>%
  unique()

df_plot_qual <- df %>%
